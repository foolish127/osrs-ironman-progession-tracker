# Ideas

Official OSRS Hiscores: For Skill XP and Boss KC.

    Endpoint: https://secure.runescape.com/m=hiscore_oldschool/index_lite.json?player=YOUR_NAME

CollectionLog.net API: This is the community standard for "Clogging." Your app will pull from their database, which is populated when you use the Runelite plugin.

    Endpoint: https://api.collectionlog.net/user/{username}

TempleOSRS API: (Optional but recommended) Great for tracking "Gains" over time (e.g., how much XP you got this week).

    Endpoint: https://templeosrs.com/api/player_info.php?user={username}
